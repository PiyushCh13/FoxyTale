# 🎮 Foxy Tale

## 📌 Overview
A charming 2D platformer where a clever fox saves the forest. Foxy Tale is a story-driven 2D platformer that follows the journey of a clever little fox through beautifully designed levels full of danger, puzzles, and mystery. Jump and explore your way through forests and outsmart enemies.

---

## 🧪 How to Play / Run the Game
1. Clone or download this repo
2. Open it in Unity
3. Press `Play` to start in the Unity Editor

> Optional: You can also download and play the standalone build from https://piyushch13.itch.io/foxy-tale
